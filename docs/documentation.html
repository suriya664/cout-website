Constructa Template Documentation

Version 1.0.0 · Updated November 7, 2025

Getting Started

1. Open any HTML file and replace placeholder copy or imagery in the `assets/images` directory.
2. Update navigation links in `includes/header.html` if you add additional pages.
3. Customize colors by adjusting CSS variables in `assets/css/style.css`.

Structure Overview

  • `assets/css/` contains core, custom, and responsive styles.
  • `assets/js/` includes main interactivity, plugin initialization, and custom utilities.
  • `includes/` stores reusable snippets for header, footer, sidebar, preloader, and modals.
  • `docs/` holds documentation, changelog, and credits.

Customization Tips

Maintain the defined spacing system to preserve design consistency. When adding new sections, apply `section-title`, `card`, and grid utility classes. Ensure hero sections maintain the minimum height and use centered headings.

Responsive Behavior

Mobile-first media queries are defined in `assets/css/responsive.css`. Adjust breakpoints or layout overrides as necessary for custom content.

Dependencies

Vendor folders are prepared for Bootstrap, AOS, Isotope, and Owl Carousel. Include necessary vendor scripts and styles in your build pipeline.
